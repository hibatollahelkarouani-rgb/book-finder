# 🌸 Book Finder

A simple book search app built with Python and Gradio.

## Features
- 🔍 Search any book by title
- 📚 Shows cover, author, publication year
- 📝 Description from Open Library

## Live Demo
👉 https://huggingface.co/spaces/hphiba/book-finder

## Tech Stack
- Python
- Gradio
- Google Books API
- Open Library API
